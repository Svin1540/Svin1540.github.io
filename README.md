# Nijisanji Guess Game

A browser-based guessing game inspired by Wordle — but instead of words, you guess **Nijisanji characters** using hints like height, debut date, birthday, and branch.

---

## ⚠️ Disclaimer

This is a fan-made project and is not affiliated with Nijisanji or Anycolor Inc.

---

## Game Modes

### Daily
Random Character every day based on day seed

### Random Mode
Random. What else should i write lmao

### Color Mode
Let's go character color (note: the color is taken from wiki)

---

## How to Play

1. Search character or just click
2. get hint
3. repeat
4. ...
5. Yeay
